## Типы коммитов

В проекте используются следующие типы сообщений коммитов:

| Тип | Назначение |
|-----|-----------|
| `feat` | Добавление новой функциональности |
| `fix` | Исправление ошибок |
| `refactor` | Рефакторинг без изменения логики |
| `perf` | Оптимизация производительности |
| `docs` | Изменения в документации |
| `test` | Добавление или изменение тестов |
| `style` | Форматирование кода (без изменения логики) |
| `chore` | Технические задачи и обслуживание проекта |
| `build` | Сборка проекта и зависимости |
| `ci` | Настройка CI/CD процессов |
